-> Steps for creating environment for testing and running the app:


For macOS/ Linux use Terminal and for Windows use PowerShell or CMD with conda support

Follow the instructions to run the app ---> 


-> $conda create -n test python=3.6.7 pip

-> $conda activate test

-> pip install -r requirements.txt

-> To run the app:

// Make sure you see a '(test)' flashing right at the start of your path in your terminal/PowerShell/CMD

-> Run 'python app.py' (without quotes)

-> Open 'localhost:5000' in the browser to run the app.
